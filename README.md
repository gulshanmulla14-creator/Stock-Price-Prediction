# Stock Price Prediction System Using Machine Learning

## Project Overview
This project predicts stock prices using historical data.  
It demonstrates **data analysis, preprocessing, machine learning, and visualization** techniques using Python.  

---

## Technologies Used
- Python
- Pandas (Data manipulation)
- NumPy (Numerical computations)
- Matplotlib / Seaborn (Data visualization)
- Scikit-learn (Machine Learning models)
- Flask ( Web interface)

---

## Features
- Collects and preprocesses historical stock data
- Implements a machine learning model to predict stock prices
- Visualizes trends and predictions
- Optional: Web interface using Flask to display predictions
